# 'The Flex'
An attribute-based responsive grid system.

## Usage

###Layout options
There are two layout options available: 'row' for horizontal content distribution, and 'col'(column) for vertical content display.

~~~~~~html
<!--Row-->
<div layout="row">
    <span>Horizontal</span>
    <span>Content</span>
</div>

<!--Column-->
<div layout="col">
    <span>Vertical</span>
    <span>Content</span>
</div>
~~~~~~
